Matlab - Authority

Matlab 全称 Matrix Laboratory.

专用于处理矩阵与数组数据.

Variables / 变量

  • Matlab 中的变量都是多维数组，与数据类型无关.

    矩阵其实本身就是用于线代数据处理的多维数组.

  • Matlab中对数字的处理与保存方式是以 long 浮点值的形式存储.

    可以使用 format 来指定显示的小数位数，而不影响存储方式.

  • 特殊常量值.

    NaN 非数字

    Pi π

    Inf 无穷

Array creation / 数组创建

  • 行向量的创建

    [1 2 3]

    []为串联运算符，可用于创建数组，逗号或者空格 用于分割各个元素

  • 多行矩阵的创建

    [1 2 3;4 5 6;7 8 9]

    ; 用于分割各行元素

  • 基于函数创建矩阵

    ones，zeros，rand来直接生成指定大小与维度的矩阵

Array Operation / 数组运算

  • 矩阵运算符

    针对于矩阵性质的运算.

  • 数组运算符 / 元素级运算

    对矩阵中各元素逐个进行运算.
    注意:

    1. 数组运算符 .^ 若改变运算变量前后次序，结果不相同.

Index of Array / 数组索引

访问数组特定元素.

  • 指定行列下标

    通过 变量名(行，列) 的方式进行索引.

    若作右值，则索引不能越界.

    若作左值，索引可以越界，编译器会自动扩大数组来容纳新的元素.

    注意:

    1. 关键字 end 表示维度下最后一个元素.

  • 线性索引

    直接指定一个下标 i .

    线性索引的情况下，数组被认为是一个长列向量(由各个列拼接而成的长列向量).

  • 多个元素的索引

    使用 : 运算符指定多个元素.

    注意:

    1. start : end 将生成一个指定范围的行向量.
    2. 可以通过 [] 进行索引.

  • 逻辑索引

    x(l) 指定 x 中的满足条件的元素，生成一个列向量.

    x 为数组，l 为与 x 同等大小的数组，l 元素非零.

    注意:

    1. 逻辑索引取 x 满足条件的元素时，将 x 视为长列向量，逐个元素的进行判断提取.

  • find 函数

    确定与指定逻辑条件相符的数组元素的索引，生成一个列向量.

    注意:

    1. 原理同 [] 以列向量逐个元素索引相同.

In series / 串联

  • [] 为串联运算符

    可串联各个元素形成数组，也可串联数组形成更大数组.

    以逗号或者空格分割表示横向串联.

    以分号分割表示纵向串联.

WorkSpace Variables / 工作区变量

从Matlab创建或者导入的变量将被包含在工作区.

String / 字符串数组

a = “…” a为字符串数组.

a只有一个元素.

  • 若””中想包含””，不再使用转义字符，而是使用双””…””来表示.

  • 拼接运算符

    可以使用+进行字符串数组内容的的首尾拼接.

  • [] - 字符串数组的串联

    字符串数组的串联将等同于一般元素的串联，将构成多行多列的矩形数组.

    并且可以通过 strlength(A) 求得各个元素的字符串长度，以矩阵形式输出.

Char / 字符数组

a = ‘…’ a为字符数组.

  • 字符在内部以数字存储，对应可使用的 ASCLL 值.、

单行字符数组

  • 字符数组每个字符为独立的元素，可以通过索引进行存取.

  • 拼接运算符.

    +不再适用于字符数组，可以相加，但是字符数组内容变为两者ASCLL的值和.

  • [] - 字符数组的串联.

    字符数组的串联将对字符数组进行首尾拼接，得到一个新的字符数组.

  • char 函数.

    可以通过 double(字符 / 字符数组)，char(数字 / 数值数组) 进行转换.

    注意:

    1. 此转换不仅仅为单个字符的转换，数组也能进行转换.

多行字符数组

多行字符数组必须满足各行字符数组长度一致，因此不足长度的需要后置位补空格.

  • char 函数.

    通过 char 可以将不等长的单行字符数组按行进行串联，后置位自动补足.

    注意:

    1. 不等长的字符数组不能按行进行串联.

Function call - 函数调用

函数名(参数列表).

  • 多参数逗号分隔，无参数不需要括号.

  • 多个返回值.

Map drawing / 绘制图

  • 二维图

    给定自变量的范围.

    给定自变量与因变量之间的关系(函数).

    plot(x,y). 绘制二线线图.

  • 三维图

    给定自变量的范围.

    给定

    surf(x,y,z). 绘制三维曲面图.

  • 标签绘制

  • 多曲线绘制

    图窗窗口中在绘制一条新的曲线时会抹除原先的曲线.

    可以通过将当前曲线添加到图窗中完成同时显示.

  • 多图绘制

    允许一个图窗中显示多个图.

Script / 脚本

  • 普通脚本

.m 文件为普通脚本文件.

注意:

  1. matlab 可用 … 表明长语句在下一行执行.

  • 实时脚本

    .mlx 文件为实时脚本文件.

  • 循环语句

    for or while.

    continue or break or return.

  • 条件语句

    if elseif else end.

    switch case otherwise.

    注意:

    1. == 若对于数组而言，不会返回 true / false，而是返回一个矩阵.
    2. 因此使用函数来进行逻辑判断 isequal isempty all any.

Help / 帮助文档

  • doc fucname.

    以文档形式打开函数文档.

  • funcame(

    填充式提示函数参数.

  • help fucname.

    命令行显示函数简明文档.

Matrix / 矩阵

Matlab 中矩阵为值为数值的矩形数组.

注意:

Matlab处理矩阵时会优先处理矩阵的列.

Matrix Creation / 矩阵创建

  • 指定明确列表创建.

    使用空格或逗号分隔行的元素。

    使用分号 ; 表示每行末尾。

    使用方括号 [ ] 将整个元素列表括起来。

  • 内置函数生成.

    zeros - 1

    ones - 0

    rand - (0,1] 均匀分布随机数

    randn (0,1] 正态分布随机数

  • 表达式创建.

    matrix = (0:9).

    创建由 0 - 9 构成的行向量.

  • 外部文件导入.

Matrix Operation / 矩阵操作

  • sum 求指定行列的和.

    sum(A) 默认求 A 矩阵列元素总和，结果为行向量.

    sum(A’)’ 可求 A 矩阵行元素总和，结果为列向量.

    sum(A,2) 第二个参数指定维度，1为按列求和，2位按行求和.

  • 子矩阵.

    A(x,y) 取 A 的子矩阵.

    x 从行的维度设置提取的内容，y 从列的维度设置提取的内容.

    1. 数值 - 提取指定行.
    2. : - 提取整个维度.
    3. start : end - 提取指定范围的内容. 提取由 start 到 end 范围且步长为1的所有元素.
    4. [1,3,2] 按指定顺序提取相应 行/列 号的内容.

    注意:

    1. 上述四种方式一定明确在何种维度下，行，列或者长列向量.

  • 删除行列元素.

    将指定元素设置为 [].

    注意:

    1. 删除元素和指定范围元素为特定值不一样，删除元素可能会导致矩阵长度不一，而发生错误.
    2. 可以通过单一下标删除一个或多个指定元素，但是其余元素将被重构为行向量.

Cell / 元胞数组

包含其他数组副本的多维数组.

  • 副本而非指针.

    即操作元胞数组元素不会对原始数组产生影响.

  • 使用 {} 进行创建和索引.

    {} 创建.

    {} + 下标 索引.

Struct / 结构体

数组元素为含字段结构体的多维数组.

  • 结构体的创建.

  • 结构体的赋值.

Function / 函数

function 关键字用于定义函数.

r - 返回值.

rank - 函数名称.

(arg) - 参数列表.

% - 第一行注释为 H1 标题行，help 时显示.

  • 匿名函数

    匿名函数可以不创建文件快速创建简单函数.

  • 一般函数

    任何非匿名函数都必须在文件中定义，且主函数与文件名相同.

    主函数 : 可供其他文件调用，最先定义.

    局部函数 : 仅供所处文件下的其他函数调用，主函数之后定义.

  • 私有函数

    私有函数位于名为 private 的文件夹中，仅供父文件夹中的函数使用.

    注意:

    1. 与私有函数同名的其他函数，在父文件夹中调用时优先搜索私有函数，以此创建特定版本的函数.

  • 全局变量

    通过 global 关键字进行声明的变量，可被多个函数共享.

    注意:

    1. 一般全局变量使用大写字符用于区分.

clc 清除命令行中所有命令

clear all 清除工作区中的所有变量

单行注释 % - %%

多行注释 %{ … %}

变量命名

区分大小写，字母开头完事

数据类型

  • 数字

  • 字符与字符串

    ‘ ‘单引号括起来的为字符串

  • 矩阵

    A = [1 2 3;4 5 6;7 8 9]
    % 转置
    B = A’
    % 竖着拉伸 以列为基准转换为列向量
    C = A(:)
    % 求逆矩阵 方阵
    D = inv(A)
    % 生成全是0的矩阵 （行数，列数，维度）
    E = zeros(10,5,3)
    % rand(0-1之间，随机均匀分布,mn的随机矩阵),randn(0-1之间，正态分布,mn的随机矩阵) randi(iMax,[m,n]) {(0,IMax]之间左开右闭，m*x随机整数矩阵}
    E(:,:,1) = rand(10,5)
    E(:,:,2) = randn(10,5)
    E(:,:,3) = randi(5,10,5)

  • 元胞数组

    cell(行数,列数)

    从1开始索引

    类似于数组，内部元素可以属于不同的数据类型.

  • 结构体

    struct(键，值…)

    以键值对的方式来存数据，仍需要通过键来取值
